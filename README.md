## アプリの概要

‐投稿の編集・削除<br>

　投稿をする際に行うタグ付けで、入力するだけでなく過去に入力したものをチェックボックスとして選択できるようにした。<br>
　またこの時、投稿内容とタグのリレーションの設定に苦労しました。
