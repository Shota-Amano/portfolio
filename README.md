## 作成した背景

プログラミングのアウトプットとして、普段自分が困っていることに関連するアプリやサイトの作成をしようと思ったところ、なかなか思い浮かばず、ほかの人がどういったことで困っているのかを知りたいという目的で今回の成果物を作成しました。

　またこの時、投稿内容とタグのリレーションの設定に苦労しました。
